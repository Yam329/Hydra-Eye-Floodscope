# Hydra-Eye-Floodscope
Geospatial data exploration application with voice/text search using Google Speech Recognition, KeyBERT, Nominatim, and Overpass APIs. Visualizes nearby geographic features like buildings and water bodies on interactive Folium maps with a Tkinter-based GUI for easy geospatial analysis.
This project is a geospatial data exploration application developed using Python and Tkinter. It supports voice and text-based location search using Google Speech Recognition API. The application fetches geographic data such as buildings, waterbodies, and trees using OpenStreetMap APIs and visualizes them on interactive Folium maps.

## Features
- Voice and text input
- Geolocation using Nominatim API
- Waterbody and building detection
- Interactive flood risk map
- Tkinter GUI

## Technologies Used
- Python
- Tkinter
- Folium
- SpeechRecognition
- OpenStreetMap APIs
